تقدیر و تشکر
--------------------------------------------------------------------------------
این برنامه از MasterDnsVPN به عنوان کلاینت VPN استفاده می‌کند.

  MasterDnsVPN توسط MasterkinG32 (Amin Mahmoudi) توسعه داده شده است
  گیت‌هاب:  https://github.com/masterking32/MasterDnsVPN

  تمام اعتبار کلاینت VPN متعلق به MasterkinG32 و مشارکت‌کنندگان است.
  این ابزار KevinNet یک برنامه جانبی برای راحت‌تر کردن پیدا کردن
  DNS Resolver های مناسب و راه‌اندازی MasterDnsVPN است — مخصوصاً برای
  کاربران داخل ایران.

  از MasterkinG32 برای ساخت و انتشار MasterDnsVPN به صورت متن‌باز سپاسگزاریم.

دانلود کلاینت MasterDnsVPN
--------------------------------------------------------------------------------
قبل از کامپایل، باید فایل اجرایی MasterDnsVPN مناسب سیستم‌عامل خود را
از صفحه رسمی دانلود کنید:

  همه نسخه‌ها:
    https://github.com/masterking32/MasterDnsVPN/releases

  لینک مستقیم دانلود:

    ویندوز (AMD64 / x64) — برای اکثر کامپیوترها:
      https://github.com/masterking32/MasterDnsVPN/releases/latest/download/MasterDnsVPN_Client_Windows_AMD64.zip

    ویندوز (ARM64) — برای Surface Pro X و لپ‌تاپ‌های Snapdragon:
      https://github.com/masterking32/MasterDnsVPN/releases/latest/download/MasterDnsVPN_Client_Windows_ARM64.zip

    مک (AMD64 / Intel):
      https://github.com/masterking32/MasterDnsVPN/releases/latest/download/MasterDnsVPN_Client_MacOS_AMD64.zip

  پس از دانلود، فایل ZIP را باز کنید و فایل اجرایی را تغییر نام دهید:
    ویندوز  → MasterDnsVPN.exe
    مک      → MasterDnsVPN

  هر دو فایل را در پوشه Build کنار kevinnet.py قرار دهید.
  این فایل‌ها به صورت خودکار داخل برنامه کامپایل‌شده قرار می‌گیرند.

  توجه: نسخه رسمی ARM64 برای مک از MasterkinG32 موجود نیست.
  برای مک Apple Silicon، از نسخه AMD64 استفاده کنید — زیر Rosetta 2 کار می‌کند.

--------------------------------------------------------------------------------

قبل از کامپایل، مطمئن شوید همه این فایل‌ها در یک پوشه هستند:

  kevinnet.py     ← برنامه اصلی
  MasterDnsVPN                ← فایل اجرایی MasterDNS برای مک/لینوکس
  MasterDnsVPN.exe            ← فایل اجرایی MasterDNS برای ویندوز
  build_mac_universal.sh      ← اسکریپت کامپایل برای مک

================================================================================
  کامپایل روی macOS (Universal — هم Intel هم Apple Silicon)
================================================================================

پیش‌نیازها:
  - مک Apple Silicon (M1, M2, M3, M4)
  - Xcode Command Line Tools
  - Homebrew
  - اتصال به اینترنت (فقط برای اولین بار)

مرحله ۱ — نصب Xcode Command Line Tools (اگر نصب نیست)

  ترمینال را باز کنید و این دستور را اجرا کنید:
    xcode-select --install

  یک پنجره نصب باز می‌شود. روی Install کلیک کنید و صبر کنید (~۵ دقیقه).
  اگر قبلاً نصب کرده‌اید این مرحله را رد کنید.

مرحله ۲ — نصب Homebrew (اگر نصب نیست)

  در ترمینال اجرا کنید:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  دستورالعمل‌های روی صفحه را دنبال کنید. در پایان دو دستور به شما می‌دهد
  که آن‌ها را هم اجرا کنید (برای اضافه کردن brew به PATH).

مرحله ۳ — نصب Python و Tkinter از طریق Homebrew

  در ترمینال اجرا کنید:
    brew install python@3.12 python-tk@3.12

  این دستور Python بومی Apple Silicon با پشتیبانی Tkinter نصب می‌کند.
  Python پیش‌فرض مک شامل Tkinter نمی‌شود.

مرحله ۴ — ساخت محیط مجازی (venv)

  به پوشه Build خود بروید:
    cd ~/Downloads/MasterDNSVPN

  venv را با Python نصب‌شده از Homebrew بسازید:
    /opt/homebrew/opt/python@3.12/bin/python3.12 -m venv venv

  آن را فعال کنید:
    source venv/bin/activate

  باید (venv) را در ابتدای خط فرمان ترمینال ببینید.

مرحله ۵ — نصب پکیج‌های Python داخل venv

  با venv فعال، اجرا کنید:
    pip install dnspython pyinstaller pillow

  منتظر نصب همه پکیج‌ها بمانید. این کار فقط یک بار لازم است.

مرحله ۶ — اجرای اسکریپت کامپایل

  اسکریپت را قابل اجرا کنید (فقط دفعه اول):
    chmod +x build_mac_universal.sh

  اجرا کنید:
    ./build_mac_universal.sh

  اسکریپت به صورت خودکار:
    ۱. Rosetta 2 را بررسی می‌کند (اگر نباشد نصب می‌کند)
    ۲. Python Intel (x86_64) را پیدا یا نصب می‌کند
    ۳. پکیج‌ها را برای هر دو معماری نصب می‌کند
    ۴. باینری ARM64 را می‌سازد
    ۵. باینری x86_64 را می‌سازد
    ۶. هر دو را با lipo ترکیب می‌کند (Universal Binary)
    ۷. پوشه dist/ را باز می‌کند

مرحله ۷ — خروجی

  dist/KevinNet

  این یک فایل روی تمام مک‌ها اجرا می‌شود — هم Intel هم Apple Silicon.
  همین یک فایل را برای همه مشتریان مک ارسال کنید.

نکات مهم برای مک:
  - اسکریپت را از داخل venv اجرا نکنید — اسکریپت خودش venv می‌سازد.
  - اولین کامپایل ۱۰-۱۵ دقیقه طول می‌کشد (نصب Rosetta + Intel Python).
  - کامپایل‌های بعدی ۳-۵ دقیقه طول می‌کشند.
  - اگر مک پیام "cannot be opened because the developer cannot be verified"
    نشان داد، مشتری باید راست‌کلیک → Open → Open کند.

================================================================================
  کامپایل روی Windows (x64)
================================================================================

پیش‌نیازها:
  - ویندوز ۱۰ یا ۱۱
  - Python 3.10 یا جدیدتر (از python.org، نه Microsoft Store)
  - اتصال به اینترنت

مرحله ۱ — نصب Python به درستی

  Python را از این آدرس دانلود کنید: https://www.python.org/downloads/windows/
  گزینه: Windows installer (64-bit)

  مهم در هنگام نصب:
    - تیک گزینه "Add Python to PATH" را بزنید ← این خیلی مهم است
    - روی "Install Now" کلیک کنید

  اگر Python از Microsoft Store دارید، آن را پاک و از python.org نصب کنید.
  نسخه Store با PyInstaller مشکل دارد.

مرحله ۲ — ساخت محیط مجازی (venv)

  Command Prompt را باز کنید (جستجوی "cmd" در منوی استارت):
    cd C:\Users\YourName\Downloads\MasterDNSVPN

  venv را بسازید:
    python -m venv venv

  آن را فعال کنید:
    venv\Scripts\activate

  باید (venv) را در ابتدای خط فرمان ببینید.

مرحله ۳ — نصب پکیج‌های Python

  با venv فعال، اجرا کنید:
    pip install dnspython pyinstaller pillow

مرحله ۴ — اجرای اسکریپت کامپایل

  روی build_windows.bat دابل‌کلیک کنید

  یا از Command Prompt که باز است:
    build_windows.bat

  اسکریپت به صورت خودکار:
    ۱. نصب Python را بررسی می‌کند
    ۲. dnspython و PyInstaller را نصب/آپدیت می‌کند
    ۳. EXE را با MasterDnsVPN.exe داخلش می‌سازد
    ۴. فایل‌های موقت را پاک می‌کند
    ۵. پوشه dist\ را باز می‌کند

  توجه: به عنوان Administrator اجرا نکنید. PyInstaller حالت ادمین را مسدود می‌کند.
  فقط دابل‌کلیک معمولی کنید.

مرحله ۵ — خروجی

  dist\KevinNet.exe

  این فایل را برای مشتریان ویندوز ارسال کنید.
  نیازی به نصب Python ندارند.

نکات مهم برای ویندوز:
  - Windows Defender یا آنتی‌ویروس ممکن است EXE را مشکوک تشخیص دهد.
    این یک False Positive رایج با PyInstaller است.
    پوشه dist\ را به whitelist آنتی‌ویروس اضافه کنید.
  - اگر پنجره کامپایل سریع بسته شد، Command Prompt را دستی باز کنید،
    به پوشه بروید و build_windows.bat را اجرا کنید تا خطا را ببینید.

================================================================================
  کامپایل روی Windows (ARM64) — برای Surface Pro X و لپ‌تاپ‌های Snapdragon
================================================================================

مرحله ۱ — دانلود Python ARM64 از python.org
  https://www.python.org/downloads/windows/
  گزینه: Windows ARM64 installer

  در یک پوشه جداگانه نصب کنید، مثلاً: C:\Python312-arm64\

مرحله ۲ — ساخت venv با Python ARM
  C:\Python312-arm64\python.exe -m venv venv_arm
  venv_arm\Scripts\activate

مرحله ۳ — نصب پکیج‌ها و کامپایل
  pip install dnspython pyinstaller pillow
  python -m PyInstaller --onefile --windowed --name "KevinNet_ARM64" --add-data "MasterDnsVPN.exe;." --add-data "MasterDnsVPN;." --clean kevinnet.py

خروجی: dist\KevinNet_ARM64.exe

================================================================================
  نحوه استفاده از برنامه (برای توضیح به مشتری)
================================================================================

برنامه DNS Resolver هایی که با MasterDnsVPN در ایران کار می‌کنند پیدا می‌کند:

  مرحله ۱: اسکن سریع — حدود ۶۰,۰۰۰ آدرس IP ایرانی را تست می‌کند
             تا Resolver های DNS باز پیدا کند (~۵-۸ دقیقه)

  مرحله ۲: تست عمیق ۶ معیاره — هر Resolver زنده را امتیازدهی می‌کند:
             NS→A، TXT، RND، DPI، EDNS، NXD
             همه Resolver ها نمایش داده می‌شوند

  مرحله ۳: تأیید E2E — Resolver ها را از طریق باینری واقعی MasterDNS
             تست می‌کند — تنها فیلتر واقعی است

  ذخیره: client_config.toml و client_resolvers.txt را در پوشه‌ای به نام
          کشور کنار خود برنامه ذخیره می‌کند، به همراه MasterDnsVPN

  اتصال: MasterDnsVPN را از پوشه ذخیره‌شده در ترمینال راه‌اندازی می‌کند

نحوه استفاده مشتری:
  ۱. دامنه تانل را وارد کنید (مثلاً v.example.com)
  ۲. کلید رمزنگاری ۳۲ کاراکتری را وارد کنید
  ۳. نام کشور را وارد کنید (مثلاً Iran)
  ۴. روی شروع اسکن کلیک کنید — حدود ۱۰-۱۵ دقیقه صبر کنید
  ۵. روی ذخیره فایل‌ها کلیک کنید
  ۶. روی اتصال MasterDNSVPN کلیک کنید

================================================================================
  رفع مشکلات رایج
================================================================================

مک: python-tk بعد از brew install پیدا نمی‌شود
  دستور: brew reinstall python-tk@3.12
  یا:    brew install tcl-tk

مک: برنامه "damaged" نشان می‌دهد یا باز نمی‌شود
  در ترمینال اجرا کنید: xattr -cr /path/to/KevinNet
  این خاصیت قرنطینه را حذف می‌کند.

ویندوز: "pip is not recognized"
  Python به PATH اضافه نشده. Python را دوباره نصب کنید و
  تیک "Add Python to PATH" را بزنید.

ویندوز: کامپایل با خطای "access denied" متوقف شد
  به عنوان Administrator اجرا نکنید.
  مطمئن شوید آنتی‌ویروس PyInstaller را مسدود نکرده.

ویندوز: EXE توسط آنتی‌ویروس flag شد
  این یک False Positive رایج با PyInstaller است.
  فایل را به استثناهای آنتی‌ویروس اضافه کنید.

عمومی: برنامه باز می‌شود ولی اسکن ۰ نتیجه می‌دهد
  - دامنه را بررسی کنید
  - مطمئن شوید سرور MasterDNS شما در حال اجراست
  - Timeout را به ۵ ثانیه افزایش دهید
  - Concurrency را به ۵۰ کاهش دهید اگر اتصال کند است
